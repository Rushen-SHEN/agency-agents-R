# 社媒运营 · Strategy Dispatch

本目录定义两条常设 **Task Force**，成员**允许交叉**（共享「内容工厂」「品牌护栏」「数据复盘」「趋势情报」四条职能）。

## 口语别名（你可以这么说）

| 你说 | 等于 | 平台 |
|------|------|------|
| **社媒一部** / 「跑一部」 | TF-01 | LinkedIn + X（Twitter） |
| **社媒二部** / 「跑二部」 | TF-02 | 抖音 + 小红书 + 微信公众号 |

对模型下指令时用 **「一部」「二部」** 即可，不必背英文代号；若需要严谨对齐 RACI，再补一句「按 `strategy/strategy-dispatch/social-media/`」。

### 激活话术示例（自然说法）

**一部（领英 + X）**

- 「帮我**跑一下社媒一部**，这周想推的是：[一句话目标 / 活动名]。你按仓库里社媒 dispatch 的 playbook 来，缺什么信息再问我。」
- 「**社媒一部**开工：先做流量密码对标，再出下周日历，最后给一版发帖草稿。」

**二部（抖音 + 小红书 + 公号）**

- 「**社媒二部**跑起来：热点侧写三条可拍选题，再对齐公众号一篇长文，缺品牌禁区你问 Brand Guardian 那套。」
- 「跑**二部**，从起号检查清单开始，我们已有账号：[平台列表]。」

**两队都要提一嘴时**

- 「社媒一部、二部**叙事别打架**，先对齐一条主轴再各写各台。」

## 共享成员（两队皆可抽调）

以下角色不按平台拆分编制，按**排期与 RACI**进入某一队当周 sprint：

- **Content Creator** — 多格式文案与脚本母本、跨平台改写 brief。
- **Trend Researcher** — 全球/英文语境趋势、竞品内容结构、社交舆情与搜索信号。
- **Brand Guardian** — 调性、视觉与叙事红线；发布前统一签核。
- **Analytics Reporter** — 指标定义、报表模板、实验读数与周报复盘。
- **Growth Hacker**（可选）— 与付费增长、落地页、裂变实验衔接时编入 TF-01 或 TF-02。
- **Legal Compliance Checker**（可选）— 强监管行业（医疗、金融等）内容合规模块。
