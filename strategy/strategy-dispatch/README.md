# Strategy Dispatch

**Strategy Dispatch** 存放「跨 NEXUS 阶段、以业务战线编组」的作战包：在标准 `playbooks/phase-*.md` 之外，为特定职能（此处为**社媒运营**）定义 **Task Force 编制、R&R、工作流与工具技能基线**。

| 文档 | 说明 |
|------|------|
| [社媒运营总入口](./social-media/README.md) | 两条战线索引、共享成员、与主策略文档的关系 |
| [账号运营 Playbook](./social-media/playbook-account-operations.md) | 从起号、流量研究、排期、创作、发布到复盘的全流程 |
| [Task Force 01 — 专业社交双栈](./social-media/task-force-01-linkedin-twitter.md) | 领英 + X（推特） |
| [Task Force 02 — 华语增长三角](./social-media/task-force-02-douyin-xhs-wechat.md) | 抖音 + 小红书 + 微信公众号 |

与 NEXUS 主线的关系：社媒 Dispatch 可在 **Phase 3（筹备内容）**、**Phase 5（上线发声）**、**Phase 6（持续运营）** 任意阶段并行挂载；详细阶段门闸仍以 `strategy/playbooks/` 与 `strategy/nexus-strategy.md` 为准。

**口语**：**社媒一部** = 领英 + X；**社媒二部** = 抖音 + 小红书 + 微信公众号（见 `social-media/README.md` 激活话术）。
